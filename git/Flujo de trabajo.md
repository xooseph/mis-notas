El repositorio local está compuesto por 3 árboles administrados por git. El primero es el **Directorio de trabajo** (se ubican todos los archivos), el segundo es el **Index** (actúa como zona intermedia) y el último es el **HEAD** que apunta al último commit realizado.

![[Pasted image 20260105204000.png]]

Adicionalmente hay una cuarta zona, la cual es en un servidor (en la nube), la cual es opcional y se envía el **HEAD** con el comando *git push*.
