Es un sistema de [[control de versiones]] [[distribuido]] de código abierto y libre.

Creado por Linus Torvalds.

Se utiliza para manejar proyectos, principalmente de software.

Guarda una *snapshot*, es decir, *git* almacena una referencia a la versión anterior.

Permite trabajar *offline*.
