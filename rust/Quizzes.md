1. **¿Cuál es la herramienta de la línea de comandos para administrar las versiones de Rust en la compu?**
		rustup

2. **Cada programa ejecutable de Rust debe contener una función con el nombre:**
		main

3. **¿Qué hace el comando `rustc hello.rs` en la línea de comandos?**
		`rustc` genera un binario ejecutable llamado `hello`
		
4. **¿Qué NO hace el comando `cargo run`?**
		Cargo observa los cambios en los archivos y re-ejecuta el binario con los cambios nuevos.
