**Rustup** es la herramienta de la línea de comandos que se utiliza para instalar Rust y sus distintas versiones.

Con `rustup doc` puedo checar la documentación de manera offline.

La función `main` es necesaria para cada ejecutable.

`rustfmt` se encarga de reformatear el código siguiendo las convenciones de la comunidad.
	`rustfmt` es el comando para un control más detallado y `cargo-fmt` entiende las convenciones de un proyecto que usa Cargo.
