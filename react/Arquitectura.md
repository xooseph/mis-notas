La arquitectura define el diseño de las estructuras y los elementos del programa, así como la forma en que deben ser reutilizados.

El código se puede agrupar en funciones, objetos, modelos, vistas y controladores, etc.

React propone el uso de [[componentes]] como elementos independientes con los que se contruye la interfaz.
