React está basado en que la definición de las vistas y sus actualizaciones se hagan de forma [[declarativa]] y no [[imperativa]].

No se ven los cambios explícitamente en la UI.

### Ventajas

- Por lo general requiere de menos código
- Usar la funcionalidad sin entender a detalle
- La optimización de ejecución en el código se delega a React
- Código expresivo

### Desventajas

- Díficil de entender los detalles
- Díficil de debuggear
- Complejidad de entender el manejo de estado
